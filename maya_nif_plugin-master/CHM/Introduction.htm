Introduction

The current Maya plug-in compiled for Maya version 6.5, 7.0, 8.0, and 8.5, for Windows only. It should also be possible to compile it from source for other platforms and versions, but no binaries for these been submitted so far. The plug-in enables the importing and exporting of NIF files, including skins bound to bones, but does not deal with animation, cameras, lights, LOD levels, particles, and other advanced features of the NIF format.
