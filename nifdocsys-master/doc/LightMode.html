LightMode

An unsigned 32-bit integer, describing how vertex colors influence lighting.

Choices

Number Name Description
0 LIGHT_MODE_EMISSIVE Emissive.
1 LIGHT_MODE_EMI_AMB_DIF Emissive + Ambient + Diffuse. (Default)

Storage Type

unsigned int
