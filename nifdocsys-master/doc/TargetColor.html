TargetColor

Used by NiPoint3InterpControllers to select which type of color in the controlled object that will be animated.

Choices

Number Name Description
0 TC_AMBIENT Control the ambient color.
1 TC_DIFFUSE Control the diffuse color.
2 TC_SPECULAR Control the specular color.
3 TC_SELF_ILLUM Control the self illumination color.

Storage Type

unsigned short

Found In

  • NiPoint3InterpController
