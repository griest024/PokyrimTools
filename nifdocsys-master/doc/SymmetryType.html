SymmetryType

Determines symetry type used by NiPSysBombModifier.

Choices

Number Name Description
0 SPHERICAL_SYMMETRY Spherical Symmetry.
1 CYLINDRICAL_SYMMETRY Cylindrical Symmetry.
2 PLANAR_SYMMETRY Planar Symmetry.
