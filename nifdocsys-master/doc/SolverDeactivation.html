SolverDeactivation

A list of possible solver deactivation settings. This value defines how the
solver deactivates objects. The solver works on a per object basis.
Note: Solver deactivation does not save CPU, but reduces creeping of
movable objects in a pile quite dramatically.

Choices

Number Name Description
0 SOLVER_DEACTIVATION_INVALID Invalid
1 SOLVER_DEACTIVATION_OFF No solver deactivation
2 SOLVER_DEACTIVATION_LOW Very conservative deactivation, typically no visible artifacts.
3 SOLVER_DEACTIVATION_MEDIUM Normal deactivation, no serious visible artifacts in most cases
4 SOLVER_DEACTIVATION_HIGH Fast deactivation, visible artifacts
5 SOLVER_DEACTIVATION_MAX Very fast deactivation, visible artifacts

Storage Type

byte
